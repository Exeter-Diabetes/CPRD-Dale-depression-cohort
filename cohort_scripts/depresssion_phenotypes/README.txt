# The following phenotypes have been generated:

- PHQ at diagnosis: These values were defined as "at diagnosis" if they were received within 30 days before and 7 days after the individual's depression index date. All scores were converted
into the categorical format.

- TRD phenotype: Based on the work of Fabbri et al 2021 (https://pubmed.ncbi.nlm.nih.gov/33753889/).

- Antidepressant coverage: Generates antidepressant coverage over given set windows since follow up using percent days covered (PDC), according to turkmen et al 2025 (https://pubmed.ncbi.nlm.nih.gov/40696379/).
